# React Material Design Lite
Example of a [React](https://facebook.github.io/react/) application using [Material Design Lite](http://www.getmdl.io/).

## Getting Started

### Prerequisites
Install NodeJS https://nodejs.org/

### Download / Clone

Clone the repo using Git:

```bash
git clone https://github.com/HenriqueLimas/react-mdl.git
```

Alternatively you can [download](https://github.com/HenriqueLimas/react-mdl/archive/master.zip)
this repository.

### Build

Install the necessary dependencies, from the root of the project:
```bash
npm install -g webpack && npm install
```
